Expense Tracker (Трекер расходов)
Автор: Анастасия Байшева 

Описание программы
Expense Tracker — это GUI-приложение для отслеживания личных расходов. Программа позволяет добавлять записи о расходах с указанием суммы, категории и даты, а также фильтровать и подсчитывать общую сумму расходов за выбранный период.

Функциональность
Добавление расходов с указанием суммы, категории и даты
Просмотр всех расходов в удобной таблице
Подсчет общей суммы расходов
Фильтрация по категории
Фильтрация по диапазону дат
Валидация вводимых данных
Сохранение данных в JSON-файл
Как использовать
Запуск приложения
Убедитесь, что у вас установлен Python 3.6 или выше
Скачайте файл main.py
Запустите приложение командой:
Добавление расхода
Введите сумму в поле "Сумма"
Выберите категорию из выпадающего списка
Введите дату в формате ДД.ММ.ГГГГ
Нажмите кнопку "Добавить расход"
Фильтрация данных
По категории: выберите категорию из выпадающего списка в блоке "Фильтры"
По дате: введите начальную и/или конечную дату и нажмите "Применить"
Сброс фильтров: нажмите кнопку "Сбросить"

Примеры использования
Пример 1: Добавление расходов
Сумма: 500, Категория: еда, Дата: 15.10.2023
Сумма: 100, Категория: транспорт, Дата: 16.10.2023
Сумма: 1000, Категория: развлечения, Дата: 17.10.2023
Пример 2: Фильтрация
Выберите категорию "еда" — отобразятся только расходы на еду
Установите период с 15.10.2023 по 16.10.2023 — отобразятся расходы за эти даты
